# تحلیل داده‌های 

این پروژه به بررسی و تحلیل داده‌های مربوط به عناوین خبری می‌پردازد. مراحل انجام پروژه شامل موارد زیر است:

- پاک‌سازی داده‌ها
- تحلیل زمانی
- تحلیل احساسات اخبار
- مصورسازی داده‌ها

## فایل‌ها

- `final_project.ipynb`: نوت‌بوک اصلی تحلیل داده
